# Image Gallery

## Overview
This project is a responsive image gallery developed as part of an internship task. It demonstrates the use of HTML, CSS, and JavaScript to build a clean user interface with basic interactivity and a professional lightbox view.

The focus of the project is clarity, usability, and proper frontend fundamentals rather than excessive features.

---

## Task Requirements
The following requirements were implemented as part of the task:

- Design an image gallery using HTML and CSS
- Display images in a responsive layout
- Add hover effects for better user experience
- Use JavaScript for image interaction
- Implement a lightbox view with next and previous navigation
- Ensure the layout works across different screen sizes

---

## Features
- Responsive image grid layout
- Category-based image filtering
- Uniform image card heights
- Hover animation on gallery images
- Lightbox modal with next and previous navigation
- Clean and minimal UI design

---

## Technologies Used
- HTML5
- CSS3
- Bootstrap 5 (via CDN)
- JavaScript
- jQuery (via CDN)
- Font Awesome (via CDN)

---

## Image Source
All images used in this project are publicly available image URLs from **Pexels** and are used only for demonstration purposes.  
No external API integration is used.

## How to Run
1. Download or clone the project folder.
2. Open the `index.html` file in any modern web browser.
3. No additional setup or configuration is required.

This project is intentionally kept simple to clearly demonstrate frontend concepts such as layout structuring, styling, event handling, and basic JavaScript logic. The implementation avoids unnecessary complexity to maintain readability and reliability.
