# Image Gallery

## Overview
This project is a responsive image gallery created as part of an internship task. It demonstrates the use of HTML, Bootstrap, CSS, and JavaScript to build a clean image grid with a professional lightbox experience.

Bootstrap utility classes are used for image styling to ensure consistency and reliability.

---

## Features
- Responsive image grid using Bootstrap
- Category-based image filtering
- Uniform image heights
- Rounded images with hover box-shadow
- Professional lightbox using LightGallery
- Zoom and thumbnail support

---

## Technologies Used
- HTML5
- CSS3
- Bootstrap 5 (CDN)
- JavaScript
- jQuery (CDN)
- LightGallery (CDN)
- Font Awesome (CDN)

---

## Image Source
Images are loaded using publicly available URLs from **Pexels** for demonstration purposes.

---

## How to Run
1. Download or clone the project folder.
2. Open `index.html` in a modern web browser.
3. No additional setup is required.

---

## Notes
Bootstrap utilities are used for image layout and hover effects, while custom CSS is limited to typography and basic layout styling.
