# Calculator

## Overview
This project is a responsive calculator developed as part of an internship task. It uses Bootstrap for layout, theming, and responsiveness, while JavaScript handles calculation logic and user interaction.

A global theme toggle allows switching between light and dark modes.

---

## Features
- Basic arithmetic operations (+, −, ×, ÷)
- Clear/reset functionality
- Responsive UI built with Bootstrap
- Keyboard input support
- Global light and dark theme switching
- Clean and professional interface

---

## Technologies Used
- HTML5
- CSS3 (Typography only)
- Bootstrap 5 (CDN)
- JavaScript
- Font Awesome (CDN)

---

## How to Run
1. Download or clone the project.
2. Open `index.html` in any modern browser.
3. Use the calculator via buttons or keyboard.
4. Change theme from the top-right menu.

---

## Notes
Bootstrap’s built-in theming system (`data-bs-theme`) is used to ensure consistent light and dark modes across the entire page without custom color styling.
