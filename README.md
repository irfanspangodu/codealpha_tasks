# Frontend Development Tasks

## Overview
This repository contains multiple frontend development tasks completed as part of an internship. Each task focuses on building user interfaces using HTML, Bootstrap, CSS, and JavaScript, with an emphasis on clean structure, responsiveness, and basic interactivity.

---

## Task 1: Image Gallery
A responsive image gallery that displays images in a grid layout with filtering and a professional lightbox view.

### Features
- Responsive image layout using Bootstrap
- Category-based image filtering
- Hover effects and smooth transitions
- Lightbox image preview using LightGallery
- Consistent image sizing

---

## Task 2: Calculator
A basic calculator that performs standard arithmetic operations with a clean and responsive user interface.

### Features
- Arithmetic operations (+, −, ×, ÷)
- Display screen for input and output
- Clear/reset functionality
- Keyboard input support
- Global light and dark theme toggle using Bootstrap

---

## Task 3 : Music Player
A simple music player that allows users to play audio files and control playback.

### Features
- Play, pause, next, and previous controls
- Display of song title and artist
- Progress bar with seek functionality
- Volume control
- Playlist support with autoplay

---

## Technologies Used
- HTML5
- CSS3 (used mainly for typography and minor styling)
- Bootstrap 5 (via CDN)
- JavaScript
- jQuery (where applicable)
- Font Awesome (via CDN)
- LightGallery (for image lightbox)

---

## How to Run
1. Open the corresponding `index.html` file for each task in a modern web browser.
2. For the music player task, ensure audio files are placed in the required folder.
3. No additional setup or configuration is required.

---

## Notes
These tasks are designed to demonstrate frontend development fundamentals such as layout design, responsive UI, DOM manipulation, and basic JavaScript functionality.
