# Music Player

## Overview
This project is a simple music player built using HTML, Bootstrap, CSS, and JavaScript. It allows users to play audio files, control playback, and manage a playlist through a clean and responsive interface.

---

## Features
- Play and pause audio
- Next and previous track navigation
- Display song title and artist
- Progress bar with seek support
- Volume control
- Playlist with song selection
- Automatic playback of the next song

---

## Technologies Used
- HTML5
- CSS3 (Typography only)
- Bootstrap 5 (via CDN)
- JavaScript
- Font Awesome (via CDN)

---

## How to Run
1. Add audio files to the `assets/audio` folder.
2. Update song file names in `js/script.js` if needed.
3. Open `index.html` in any modern web browser.
4. Use the player controls to play music.

---

## Notes
This project demonstrates JavaScript audio handling, DOM manipulation, and responsive UI design using Bootstrap.
